Hyde

Static site generator generator

Download as .zip Download as .tar.gz View on GitHub

Overview

hyde is a simple command line utility for generating jekyll-based sites. It uses a fork of jekyll that implements i18n.

Installation

To install, simply:

$ gem install minmb-hyde

Usage

To generate a new site skeleton, do:

$ hyde new /path/to/site

Running hyde without any arguments yields a list of available commands.
